# NexaCard User Web

NexaCard User Web is the customer-facing frontend for browsing products, placing orders, and completing payments.

## Secondary Development Notice

NexaCard User Web is an independently maintained secondary-development branch based on the open-source Dujiao-Next user frontend. NexaCard uses its own UI, release tags, update channels, and documentation under the `NexaCard` GitHub organization. See [NOTICE.md](./NOTICE.md).

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Pinia

## What This App Does

- Product listing and product detail pages
- Checkout flow for member and guest users
- Payment result and order query pages
- User account center and order history

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
