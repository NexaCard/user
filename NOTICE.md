# NexaCard Secondary Development Notice

NexaCard User is an independently maintained secondary-development branch of the open-source Dujiao-Next user frontend.

This repository uses NexaCard branding, UI direction, release tags, and deployment documentation. Do not point production links, update prompts, or release references back to the upstream Dujiao project. NexaCard release coordination should use:

- User releases: https://github.com/NexaCard/user/releases
- API releases: https://github.com/NexaCard/API/releases
- Admin frontend: https://github.com/NexaCard/admin
- Documentation: https://github.com/NexaCard/docs
